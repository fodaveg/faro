# Faro

Tema unificado para Obsidian. Faro separa el color de la estructura: cuatro esquemas completos pueden combinarse libremente con cuatro estilos de encabezados desde Style Settings.

## Qué intenta resolver

- El panel activo, la pestaña activa y el archivo seleccionado comparten el mismo color de orientación.
- Los seis niveles de encabezado se distinguen por tamaño, color y peso, no solo por una de esas señales.
- Negritas, cursivas, enlaces, etiquetas, resaltados, código y listas tienen tratamientos semánticos diferentes.
- Una sola base mantiene navegación, tablas, propiedades y compatibilidad de plugins coherentes entre esquemas.
- Incluye estados de foco visibles y respeta `prefers-reduced-motion`.

## Instalación manual

1. Copia la carpeta `Faro` dentro de `<tu bóveda>/.obsidian/themes/`.
2. En Obsidian abre **Ajustes → Apariencia → Temas**.
3. Selecciona **Faro**.
4. Usa el esquema oscuro de Obsidian para ver la versión principal del tema.

## Ajustes opcionales

El tema funciona sin plugins y utiliza Faro + encabezados limpios por defecto. Con **Style Settings** aparece una sección `Faro` con:

- Cuatro esquemas completos: **Faro**, **Noctiluca**, **Seda** y **Marea**.
- Cuatro estructuras independientes: **Limpio**, **Bandas**, **Editorial serif** y **Líneas**.
- Un modo vivo para días en los que apetece más color.
- Línea activa, ancho de lectura, interlineado y navegación compacta.

Ejemplos: `Seda + Bandas`, `Marea + Editorial serif` o `Noctiluca + Limpio`.

## Estado
